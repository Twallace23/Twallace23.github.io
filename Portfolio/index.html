IDP Auto Labeling

Worked on a development team to create a system to alleviate the time spent labeling remittance devices. Using Donut an open source transformer and language decoder we were able to design a system that extracted the key value pairs from a pdf and then fill a json with those values. Due to access rights I am not allowed to include the work. Tech stack used, React, Python, AWS, Docker.

Movie Api

Created a movie API, allowing a user to search for a movie, that movie's poster image displayed as well certain basic information about the selected movie. Used EJS,JS,JQuery, and SQL.
